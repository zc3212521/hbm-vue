# vue-base

## 命令

### 依赖安装
```
npm install
```

### 编译开发环境代码，开启热替换服务器
```
npm run serve
```

### 编译正式环境代码
```
npm run build
```

### 运行接口mock
```
npm run mock
```

### 运行代码修复跟检查
```
npm run lint
```

### 运行单元测试
```
npm run test
```

## 目录结构

```
src
├─assets  // 资源文件
│  ├─font
│  ├─images
│  ├─js
│  └─style
├─components // 共用组件
│  ├─business
│  └─common
├─http // 请求相关
├─mock // mock后端数据
├─pages
│  ├─about
│  │  └─components
│  └─home
├─store // 状态管理
│  └─modules
└─utils // 工具类
```
